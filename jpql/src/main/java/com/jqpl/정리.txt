• JPQL
  • 테이블이 아닌 객체를 대상으로 검색하는 객체 지향 쿼리
  • SQL을 추상화해서 특정 DB SQL에 의존X
  • JPQL을 한마디로 정의하면 객체 지향 SQL

• Criteria
  • 문자가 아닌 자바코드로 JPQL을 작성할 수 있음
  • JPQL 빌더역할
  • JPA 공식 기능

  단점 : 너무 복잡하고 실용성이 없다.

• QueryDSL
  • 문자가 아닌 자바코드로 JPQL을 작성할 수 있음
  • JPQL 빌더 역할
  • 컴파일 시점에 문법 오류를 찾을 수 있음
  • 동적쿼리 작성 편리함
  • 단순하고 쉬움
  • 실무 사용 권장

• 네이티브 SQL
  • JPA가 제공하는 SQL을 직접 사용하는 기능
  • JPQL로 해결할 수 없는 특정 DB에 의존적인 기능
